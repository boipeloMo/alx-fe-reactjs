# Recipe Sharing App

A simple Recipe Sharing Application built with **React** and **Zustand** for state management.  
Users can add new recipes and view a list of all submitted recipes.

---

## 🚀 Features
- Add new recipes with a title and description
- Display a list of all recipes
- Powered by Zustand for lightweight and scalable state management
